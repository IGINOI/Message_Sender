# Introduction
The idea of this project is to implement an appointment management system based on LibreOffice Calc that authomatically sends reminders via WhatsApp. The platform choice was based on the fact that it is available to everyone and that there is not the need to be connected to internet (relevant if you don't want your data around the internet for some reasong).
